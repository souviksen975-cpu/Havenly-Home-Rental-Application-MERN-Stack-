# Havenly-Home-Rental-Application-MERN-Stack-
Havenly – Home Rental Application (MERN Stack)  Built a full-stack rental management system allowing users to list properties, make bookings, and manage reservations.
•	Developed REST APIs for property listings, bookings, and cancellations using Node.js and Express.
•	Implemented role-based authentication and optimized MongoDB schema for efficient data handling.
